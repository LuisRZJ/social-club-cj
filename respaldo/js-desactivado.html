JavaScript Desactivado

Se requiere JavaScript para acceder a esta página

Nuestro sitio web necesita JavaScript para funcionar correctamente. Parece que JavaScript está desactivado en tu navegador.

La mayoría de las funcionalidades de esta página —incluyendo el menú de navegación, botones clave y desplegables informativos— dependen del uso de JavaScript para su correcta visualización. En consecuencia, la página resulta prácticamente inutilizable si este recurso está deshabilitado, debido a la estructura y componentes implementados.

Por favor, habilita JavaScript en tu navegador para continuar

Cómo habilitar JavaScript en tu navegador
Google Chrome
  1. Abre el menú de Chrome (⋮) en la esquina superior derecha
  2. Selecciona "Configuración"
  3. Haz clic en "Privacidad y seguridad"
  4. Selecciona "Configuración del sitio"
  5. Haz clic en "JavaScript"
  6. Activa "Permitido (recomendado)"
  7. Recarga la página
Mozilla Firefox
  1. Escribe "about:config" en la barra de direcciones
  2. Acepta la advertencia de riesgo
  3. Busca "javascript.enabled"
  4. Haz doble clic en la preferencia para cambiar su valor a "true"
  5. Recarga la página
Microsoft Edge
  1. Abre el menú de Edge (...) en la esquina superior derecha
  2. Selecciona "Configuración"
  3. Haz clic en "Cookies y permisos del sitio"
  4. Selecciona "JavaScript"
  5. Activa la opción "Permitido"
  6. Recarga la página
Safari
  1. Abre el menú "Safari" en la barra superior
  2. Selecciona "Preferencias"
  3. Haz clic en la pestaña "Seguridad"
  4. Marca la casilla "Habilitar JavaScript"
  5. Cierra la ventana de preferencias
  6. Recarga la página
