Ejemplo de Overlay Independiente

¿Cómo funciona?

Esta página demuestra el uso del overlay de carga independiente. El script se encarga de:

Detección Automática

Detecta automáticamente cuando el JavaScript principal ha terminado de cargar

Efectos Modernos

Incluye efectos de aparición y desaparición con animaciones suaves

Soporte de Temas

Se adapta automáticamente al tema claro/oscuro de la página

Fácil Implementación

Solo necesitas incluir el script en tu página HTML

Configuración

Código de Implementación:

<!-- Incluir el script del overlay independiente -->
<script src="/pwa/overlay-carga-independiente.js"></script>

<!-- El overlay se crea automáticamente -->
<div class="main-content" id="mainContent">
  <!-- Tu contenido aquí -->
</div>

API Disponible:

window.independentLoadingOverlay.updateProgress(message, percentage)

Actualiza el mensaje de progreso del overlay

window.independentLoadingOverlay.hide()

Oculta el overlay con efecto de desaparición

window.independentLoadingOverlay.show()

Muestra el overlay nuevamente

window.independentLoadingOverlay.restart()

Reinicia completamente el overlay

Pruebas Interactivas

Consejo: Usa estos botones para probar las diferentes funcionalidades del overlay independiente.

Características Técnicas

Timeout de Detección

10 segundos máximo de espera para detectar el JavaScript principal

Intervalo de Verificación

Verifica cada 100ms si el sistema principal ha terminado

Efectos Visuales

Animaciones suaves con backdrop-filter y transformaciones CSS

Responsive

Completamente adaptable a todos los tamaños de pantalla
